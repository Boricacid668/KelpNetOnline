# Kelp.Net
Kelp.NetはChainerを参考にC#で実装された用の深層学習の学習向けライブラリです。


##特徴
- 線形代数を用いずにひたすらforを使って実装しているので、私のように数学の知識が乏しいプログラマーでも、動作の様子がわかります。
- Linqを使用していないのでC#以外のプログラマーの方でも、比較的読めるようになっていると思います。
